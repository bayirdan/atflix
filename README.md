# Movie App - Atflix

This template is a movie app like Netflix. It is designed as fully responsive, including mobile. Used React as JS framework.

- Profile, search bar, slide button, movie detail button, hamburger menu button are active. You can change the dark mode or light mode from the profile and hamburger menu sections.

---

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
